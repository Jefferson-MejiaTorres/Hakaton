Agregar Nueva Medición

Busca un paciente y registra sus nuevas mediciones antropométricas

¿Cómo funciona?

  • Paso 1: Busca al paciente por nombre o documento
  • Paso 2: Selecciona el paciente correcto de los resultados
  • Paso 3: Ingresa las nuevas mediciones (peso, talla, etc.)
  • Paso 4: El sistema puede generar una evaluación nutricional automática

Tip: Puedes activar la evaluación automática para obtener el diagnóstico actualizado inmediatamente después de guardar las mediciones.
