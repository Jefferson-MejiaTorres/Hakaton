El Problema

La desnutrición infantil en Norte de Santander

Situación Crítica

La desnutrición infantil es un problema de salud pública grave que afecta el desarrollo físico y cognitivo de miles de niños en Norte de Santander. La detección tardía impide acciones preventivas efectivas.

Detección Tardía

Los casos se identifican cuando ya hay daños irreversibles

Falta de Datos Integrados

Información dispersa y sin análisis predictivo

Recursos Limitados

Personal de salud sobrecargado sin herramientas de apoyo

Políticas Reactivas

Decisiones sin sustento en análisis predictivo de datos

Pregunta de Investigación

"¿Cómo puede un sistema inteligente basado en minería de datos y aprendizaje supervisado predecir tempranamente la desnutrición infantil en Norte de Santander, para apoyar acciones preventivas y orientar políticas públicas de salud?"

Nuestra Solución: SIDI

Sistema Inteligente de Detección de Desnutrición Infantil

Minería de Datos

Análisis profundo de datos sociodemográficos y de salud para identificar patrones asociados a la desnutrición infantil

Aprendizaje Supervisado

Modelos de IA (SVM, Redes Neuronales, Random Forest) para predicción temprana con alta precisión

Predicción Temprana

Anticipación de riesgos en poblaciones vulnerables para acciones preventivas oportunas

Objetivos del Proyecto

1

Estudiar datos sociodemográficos y de salud mediante minería de datos para identificar patrones asociados a la desnutrición infantil

2

Seleccionar técnicas de aprendizaje supervisado (SVM, redes neuronales, Random Forest) para definir el modelo más adecuado

3

Crear un modelo predictivo aplicando algoritmos de IA para anticipar riesgos en poblaciones vulnerables

4

Validar el sistema mediante métricas de precisión, recall y AUC con validación cruzada para garantizar confiabilidad

Herramientas TIC para el Trabajo Incluyente y Seguro

SIDI es una herramienta TIC diseñada para empoderar a los profesionales de salud

¿Cómo SIDI cumple con este pilar?

Trabajo Incluyente

  • Acceso universal: Interfaz web accesible desde cualquier dispositivo, permitiendo que profesionales en zonas rurales accedan al sistema
  • Capacitación integrada: Tutoriales y guías para usuarios con diferentes niveles de conocimiento tecnológico
  • Democratización del conocimiento: Compartir insights y mejores prácticas entre centros de salud
  • Inclusión digital: Diseño pensado para personas con diferentes capacidades y contextos socioeconómicos

Trabajo Seguro

  • Reducción de errores: El modelo de IA valida y complementa el criterio médico, minimizando diagnósticos incorrectos
  • Decisiones basadas en datos: Análisis de miles de registros para recomendaciones fundamentadas
  • Privacidad y seguridad: Protección de datos sensibles de pacientes según normativas de salud
  • Trazabilidad: Registro completo de predicciones y decisiones para auditorías y mejora continua

Eficiencia

Reduce el tiempo de análisis de casos de horas a minutos

Apoyo al Personal

Herramienta de segunda opinión que aumenta la confianza profesional

Equidad

Mismo nivel de análisis en todas las zonas de Norte de Santander

Tecnología de Vanguardia

Inteligencia Artificial al servicio de la salud pública

SVM

Support Vector Machines para clasificación de riesgo con alta precisión en escenarios de datos complejos

Redes Neuronales

Deep Learning para detectar patrones no lineales y relaciones complejas en datos de salud

Random Forest

Ensemble learning para predicciones robustas y explicables, ideal para decisiones clínicas

Validación Rigurosa

Recall

AUC-ROC
